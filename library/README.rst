Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install msa301``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/msa301-python``
-  ``cd msa301-python``
-  ``sudo ./install.sh``
